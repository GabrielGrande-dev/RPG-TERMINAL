# 📜 Chronicles: RPG Terminal

**Chronicles** é um Ambiente Virtual de Aprendizagem (AVA) imersivo, focado em mestres e jogadores de RPG de mesa. O projeto combina a robustez de um sistema de gerenciamento de banco de dados com uma estética clássica medieval.

---

## 🏛️ Sobre o Projeto

O objetivo do RPG Terminal é fornecer um "Tomo Digital" onde campanhas podem ser registradas, gerenciadas e vividas. Saímos de uma interface de "terminal hacker" para um visual de "pergaminho e ferro batido", focando na imersão dos jogadores.

### ✨ Funcionalidades Atuais
* **Autenticação via Magic Link:** Login seguro e sem senha utilizando **Supabase Auth**.
* **Gestão de Crônicas:** Mestres podem criar novas campanhas ("Crônicas") diretamente no painel.
* **Interface Imersiva:** Design medieval utilizando **Tailwind CSS** e fontes serifadas clássicas (Cinzel).
* **Banco de Dados em Tempo Real:** Estrutura pronta para gerenciar pastas, documentos e NPCs.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído com o que há de mais moderno no ecossistema web:

* **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
* **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
* **Estilização:** [Tailwind CSS](https://tailwindcss.com/)
* **Backend & Auth:** [Supabase](https://supabase.com/) (PostgreSQL)
